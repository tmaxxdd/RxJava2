# RxJava2 w Android

## Zrealizowane scenariusze
* Generowanie i filtrowanie liczb
* Asynchroniczne reagowanie na widoku z RxView
* Dynamiczne wyszukiwanie słów
* Asynchroniczna baza danychz z RxFirebase
* Walidacja hasła
